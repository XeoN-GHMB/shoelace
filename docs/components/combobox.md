## Examples

### Remote source for suggestions

You can define a remote source to retrieve a list of data.

```html preview
<sl-combobox
  class="combobox-remote"
  clearable>
</sl-combobox>

<script>
  const combobox = document.querySelector('.combobox-remote');
  const url = 'https://60db3b45801dcb0017290fdb.mockapi.io/users?name={q}';

  combobox.source = search => {
    return fetch(url.replace('{q}', search))
      .then(res => res.json())
      .then(data =>
        data.map(d => {
          return {
            text: d.name,
            value: d.id
          };
        })
      );
    };
</script>
```
